# 起動方法（フェーズ1 プロトタイプ）

このドキュメントは、本プロジェクトの Phase 1 プロトタイプを **起動するための手順** をまとめたものです。

## 1. 事前条件

- Python 3.10 以上
- 依存ライブラリ: `pygame-ce`（`requirements.txt` に記載）

## 2. 最短で起動する

プロジェクトルート（`main.py` がある場所）で以下を実行します。

```bash
python main.py
```

## 3. venv を使って起動する（推奨）

### 3.1 venv 作成

```bash
python -m venv venv
```

### 3.2 venv 有効化

Windows（PowerShell）の例：

```powershell
cd .\開発用\2D格闘ゲーム\

venv\Scripts\Activate.ps1
```

### 3.3 依存ライブラリのインストール

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 3.4 起動

```bash
python main.py
```

## 4. よくあるトラブル

### 4.1 `python` が見つからない

```bash
python --version
```

- Python が未インストールなら Python 3.10 以上をインストールしてください。
- Windows ではインストール時に **"Add Python to PATH"** を有効にすると解決しやすいです。
- `python` が通らない場合、`py -3.10 main.py` を試してください。

### 4.2 `import pygame` で失敗する

```bash
pip install -r requirements.txt
```

- venv を使っている場合は、先に venv を有効化してから実行してください。
